이 프로잭트는 한양대학교 에리카 주변 맛집 정보를 사용자가 쉽게 찾을 수 있도록 카테고리 별로 음식점을 정리한 모바일 페이지를 만드는 프로잭트다.
이때 여러 검색기능을 추가하여 원하는 음식점을 찾기 쉽게 하였다
카테고리별 검색 기능, 별점, 절렬기준 설정 등의 사용자에게 도움을 줄 수 잇는 다양한 부가기능을 넣었다.
그리고 지속가능한 사용을 위해서 '찜' 기능을 추가하였다
참여 프로그래머는 손명균, 오준경, 나세진, 박준식 이다.
오준경은 프로잭트 아이디어 제시와 UI/UX설계와 UI구현,master브렌치 제작 후 기본적인 구조를 만들었다. 저장소 생성과 관리, 시스템 구조 설계와 이슈관리를 맡았다.
손명균은 방금 SON2branch 만들어서 찜하기 기능 추가찜하기 목록만 보는 기능 추가 네이버 지도랑 연결을 하였다.
나세진은 식당 넣기,검색기능, 상세보기 클릭 기능, 리뷰 쓰기+수정, 삭제기능, 별점 추가 기능
박준식은 상세페이지 지도 이미지 추가, 카테고리 기능 구현, 랜덤 추천 기능 구현, 검색창 초기화 버튼 추가, 이미지 파일 적용 및 코드 수정, 이미지파일 스타일시트 생성 ,js css 정리
